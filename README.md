# 选择主页

选择主页,帮你选择一个适合自己的浏览器主页

欢迎大家在issues提交网站

## 项目起源

最初源于知乎上的一个回答，因为文章收录的网址有限

后来便决定搭建一个方便选择浏览器主页的网站，欢迎大家提交网站

匿名在线提交地址：https://wj.qq.com/s/2136215/21b1/
